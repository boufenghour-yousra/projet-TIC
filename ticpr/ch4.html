chapitre 4 :INTERNET ET WEB

1-World Wide Web :

Ou bien ‘Le Web’ Inventé par Tim Berners-Lee et Robert Cailliau en 1989. C’est un systéme de documents interconnectés reliés par des hyperlien et des URL . Il permet aux gens d’accéder et de partager des informations de n’importe tout dans le monde à l’aide du navigateur web .

Définition :

Web : une partie des contenus circulant sur l’Internet.

URL (pour Uniform Resource Locator) : est le nom unique de la page. décrivant l’emplacement d’une ressource, elle contient :

• L’adresse du serveur.

• l’adresse de la page sur le serveur.

exemple :click

2-Aspects Techniques du Web :

On trouve 3 mécanisme du web:

    a- URL : Le mécanisme de localisation peut faire appel au protocole DNS

    b- HTTP :

    - Paradigme client-serveur

    - Protocole de nouveau applicatif et sans état « mémoire »

    c- HTML :

    - Langage de description de « pages Web »

    -Texte, images et autres objets

    - Liens hypermédias entre les pages

    -Programmation déclarative

Page et site web :

3-Page Web :

Page Web est un document multimédia avec un langage de description HTML 1992, XHTML 2002 ou HTML5 stocké dans un fichier HTML ......, affichée par un navigateur sur internet dans une adresse URL .

-Site Web :

C’est un regroupement de pages web reliées entres elles par des liens hypertexte qui permet de définir des chemins relatifs structuré comme un disque locale .

4-Création d’une page web HTML :

    Le document HTML est un fichier texte constitué :

    - D’une syntaxe précise

    - Structuré par des balises (doit comporter au minimum ces 4 balises):

    < html> ... < /html>: Délimite le début et la fin du documen

    < head> ... < /head>: Entête du document

    < title> ... < /title>: Titre du documen

    < body> ... < /body> : Corps du document

    - Créé dans un éditeur de pages web et Interprété selon votre navigateur
